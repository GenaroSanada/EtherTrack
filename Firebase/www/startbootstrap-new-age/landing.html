Data privacy

Using keccack256 algorithm, we store hashed values of your SKUs. You are the only one that can access the real information.

Auditable

EtherTrack uses vouchers (based on Ethereum transaction) to proove that you virtualy sent SKUs

Trust worthy

Due to blockchain intrasec nature, all changes are registered on the chain. Nobody can tamper with your inventory data.

Open Source

Since this DApp is MIT licensed, you can use freely access its source code !

Base on GS1 EPC Standard

EtherTrack is based on GS1 standard for Electronic Products Code
